About

Thanks for taking a moment to view my portfolio,

I became interested in programming about around April of 2020 and began following some online courses such as Harvard's CS50 course covering topics including C, Python, Javascript and HTML/CSS. I decided to commit to making coding my full time career and enrolled in Eleven Fity Academy's 12 week immersive software development course. Throughout the course I've developed skills in C# and Object Oriented Programming and learned full-stack web development using Microsoft's .NET platform along with HTML and CSS.

My most recent professional exerience was working for the Department of Veterans Affairs adjudicating disability benefit claims. Prior to that I earned a bachelor's degree in Economics from the University of Wisconsin-Milwaukee, and also served in the Marine Corps working in the aviation field where among other duties I was responsible for calculating the weight and balance of the KC-130J aircraft to ensure the center of gravity remained within safe limits during flight. While not directly related to programming my previous work and educational experience has helped me logical reasoning skills that have proved invaluable while learning software development. You can find examples of my code on Github , and see more about my experience on linkedin or by viewing my resume. BookLover API

This is a Web API that was completed as a group project using C#, ASP.NET and Entity framework. The API stores data for books and authors and also contains endpoints for users to review books and comment on reviews. Users can create "Bookshelves" or lists of books that can be shared, and also maintain a private "To Read" list. The API also has methods which utilize the Google Books API to search for books and save them to our database.

See on Github
Etch-a-Sketch

This is an "Etch-a-sketch" website built with basic HTML, CSS and Javascript which allows users to draw pictures on a colored grid. Users can change the size and colors of the grid and download a copy of their finished picture.
